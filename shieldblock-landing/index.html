How to Install (Chrome & Edge)

1
Download the extension
Click the button above to download the .zip file.
2
Extract the zip file
Unzip the downloaded file. You should see a folder named shieldblock-extension-main.
3
Open Extensions Page
Go to chrome://extensions/ (Chrome) or edge://extensions/ (Edge)
4
Enable Developer Mode
Turn on the "Developer mode" toggle in the top right corner.
5
Load Unpacked Extension
Click "Load unpacked" and select the extracted folder.

Done! The extension should now appear with "ON" badge.

Works on: Google Chrome and Microsoft Edge only
